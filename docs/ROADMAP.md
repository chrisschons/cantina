# Roadmap

## Phase 1 (current): Foundations + MVP Core

- Monorepo and Docker baseline
- API MVP: auth, profiles, channels, messages, reactions, unread basics
- Web MVP: login, channel list, message stream, composer
- Documentation and architecture records

## Phase 2: Expanded Core

- Slash commands and custom commands
- Threads and message permalinking
- Media upload and OGP previews
- Notification modes and temporary snoozes
- Search + library ingestion baseline

## Phase 3: Platform Features

- Voice/video and screen sharing
- Plugin/module capability system
- AI Librarian metadata suggestions
- Federation exploration (post-v1)
